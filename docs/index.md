## Documentation goals

This doc set is written for three audiences:

- operators who want to install and run jobs safely
- developers who want to understand the runtime and extend it
- tool builders who want to integrate through stable inspection APIs

If you are new to the project, use this order:

1. [Installation](installation.md)
2. [Quickstart](quickstart.md)
3. [Examples Guide](examples.md)
4. [CLI Guide](cli.md)
